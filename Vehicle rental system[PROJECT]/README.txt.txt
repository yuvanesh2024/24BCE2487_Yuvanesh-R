SMART VEHICLE RENTAL MANAGEMENT SYSTEM

Technology Used:
- Java
- Maven
- MongoDB
- MongoDB Compass
- Eclipse IDE

Project Structure:
- models: Contains entity classes such as Admin, Customer, Vehicle, Rental and Payment.
- services: Contains business logic and MongoDB CRUD operations.
- database: Contains MongoDB connection class.
- utils: Contains helper classes such as IDGenerator and InputValidator.
- main: Contains VehicleRentalSystem.java which runs the application.

How to Run the Project:

1. Install Java and Maven.
2. Open Eclipse IDE.
3. Select File -> Import -> Existing Maven Projects.
4. Select the VehicleRentalSystem folder.
5. Wait for Maven dependencies to download.
6. Start MongoDB server.
7. Run VehicleRentalSystem.java as Java Application.
8. Use the console menu to register customers, login, manage vehicles, rentals and payments.

Database Name:
vehicle_rental_system

MongoDB JSON files are included separately for reference.
